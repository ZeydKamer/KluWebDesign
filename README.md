# KLU Web Design Concept 🎓

A modern, highly interactive, and accessible landing page redesign concept for Kırklareli University (KLU). 

This project aims to provide a next-generation user experience for prospective students, current students, and academicians by combining modern web technologies with a strong focus on accessibility and interactive data visualization.

## ✨ Key Features

* **🎨 Modern UI/UX Architecture:** Built with advanced CSS Flexbox and Grid layouts for a clean, professional, and highly responsive structure.
* **🌓 Seamless Dark/Light Mode:** Intelligent theme switching that remembers user preferences via `localStorage`. Custom color palettes ensure perfect readability in both modes.
* **🌍 Interactive 3D Globe (D3.js):** A dynamic, data-driven 3D globe visualizing international student demographics, built using `D3.js` and `TopoJSON`. Includes automated rotation and location highlighting.
* **♿ Accessibility First (a11y):** A dedicated accessibility panel featuring:
    * Dynamic text zooming (3 levels).
    * High Contrast mode.
    * SVG-based color blindness filters (Protanopia, Tritanopia, Monochromacy).
* **🎬 Cinematic Hero Section:** Background parallax video integration with play/pause controls for optimized user experience.
* **💡 Dynamic UI Elements:** Custom-built, non-intrusive modal pop-ups, smart event date highlighters (auto-detects past/future events), and smooth scrolling navigation.
* **📱 Fully Responsive:** Carefully crafted media queries ensure a flawless experience across mobile phones, tablets, and desktop monitors.

## 🛠️ Technologies Used

* **HTML5:** Semantic structuring.
* **CSS3:** Custom variables (theming), advanced animations, CSS Grid/Flexbox, and backdrop-filters (glassmorphism).
* **Vanilla JavaScript (ES6+):** DOM manipulation, local storage management, and custom interactivity without relying on heavy frameworks.
* **D3.js (v3) & TopoJSON:** For rendering and animating the interactive 3D orthographic map.
* **FontAwesome (v6):** Scalable vector icons.
